# 项目介绍

LoRaWAN Specification 是 LoRa 联盟规范的核心协议。

本项目是 V1.1 的中文译本。出于个人学习梳理的目的进行整理。LoraWAN 规范原版权声明见原文第25章。

初始版本参照项目：twowinter 的 LoRaWan 规范中文化项目。原项目 v1.0.2 已正式发布。V1.1 翻译整理中。

由于目录层级结构组织、阅读习惯等差异较大，没有采用 Fork 的方式，而是参照其最新版本（2019.4），根据 1.1 英文原文重新翻译整理。

建议对照英文原文阅读，并欢迎通过 提交Issue 提出改进意见、揪出翻译中存在的问题。

同时，附录中整理了一些 LoRaWAN 相关的学习资料供参考。

# 一些术语的翻译说明：

  • Join：翻译为“入网”或“连接”，暂未完全统一
  • SHALL, MUST ... 等限定词，为保证准确，翻译为中文后保留原英文单词。
  • 信道、频道，本文中含义相同。均指 frequecy channel。
  • Data Rate: 翻译为数据速率、速率
  • MAC Command 翻译为 Mac 指令或命令
